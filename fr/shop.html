Se connecter avec Minecraft


Pour vous connecter au site web, lancez simplement Minecraft avec la dernière version de LabyMod et cliquez sur votre tête dans le menu principal de Minecraft ou dans les paramètres LabyMod.




Si vous rencontrez des problèmes pour démarrer Minecraft avec LabyMod ou si vous ne trouvez pas la tête dans votre menu principal, assurez-vous que vous avez la dernière version de LabyMod.

Veuillez prendre en compte que vous devez avoir acheté Minecraft Java Edition pour vous inscrire.
